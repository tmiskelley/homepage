Programming


Art through logic

While programming is an intensely logical discipline, many programmers (myself included) find it very creatively stimulating. Much like a painter uses a brush on a blank canvas, so to do programmers use keystrokes to bring their ideas to life on the screen. Programming, though, is an umbrella term which encapsulates a huge variety of disciplines. Below is a brief overview of the fundamentals.


  • Languages

    There are numerous programming languages, each with their unique syntax and use cases. Webpages like this one, for example, use Javascript for their functionality. Other popular languages include Python, Java, C++, and many, many more.

  • Variables and datatypes

    Regardless of which language you use, all programs use variables to store and manipulate data. Data types, such as integers, strings, and booleans, define the kind of information a variable can hold.

  • Logical operation

    Programs control the flow of execution through logical conditions (if statements), loops (for, while), and branching. This enables the computer to make decisions and repeat tasks as necessary.

  • Algorithims and problem solving

    Programming involves creating algorithms, step-by-step procedures for solving problems. Strong problem-solving skills are crucial for effective programming.
